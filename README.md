# Night Driver: Neon Rush (estilo NFS2 retro)

Ahora sí: esto es un **programa de escritorio real** (no solo abrir `index.html`).
Está hecho en **Python + Pygame** y busca un estilo arcade retro tipo NFS clásico:
carretera pseudo-3D, tráfico, curvas dinámicas, nitro y HUD en vivo.

## Requisitos

- Python 3.10+
- Dependencias de `requirements.txt`

## Instalación

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Ejecutar

```bash
python3 main.py
```

## Controles

- `A/D` o `←/→`: girar
- `W/S` o `↑/↓`: acelerar/frenar
- `ESPACIO`: usar nitro
- `R`: reiniciar después del choque
